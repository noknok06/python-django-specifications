# 規格変更通知アプリ

<img width="1200" alt="image" src="https://github.com/user-attachments/assets/99631c52-cfa5-4c24-992f-c37bf0eca64d">

卸⇔得意先の業務を意識したツールです。

コロナ化以降に商品の規格変更が頻発しています。仕入先より商品の規格変更が発生した場合に、

商品を登録することで、売上を上げている得意先にメール通知します。

## 基本機能

* 管理機能(CRUD)
* 規格変更登録
* メール通知
* 履歴照会
